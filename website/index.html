Installation Tutorials

Diagram pictures and setup walkthrough for each package

Starter Installation Tutorial

Best for small coverage setup (1 Year License | 50 Meters).

  1. Mount the Starter unit near your modem with stable power.
  2. Connect WAN/LAN cable and verify portal access from your phone.
  3. Set SSID, password, and bandwidth limit in setup panel.

Professional Installation Tutorial

Recommended for wider service area (3 Years License | 100 Meters).

  1. Install central router point and connect Professional unit.
  2. Enable voucher/telegram monitoring and test payment workflow.
  3. Run signal and speed checks in peak-hour conditions.

Enterprise Installation Tutorial

For high-capacity deployment (Lifetime License | 250 Meters).

  1. Deploy main unit plus access points based on coverage map.
  2. Apply anti-lag and user limit profiles before launch.
  3. Perform full activation test: payment, voucher, and portal login.

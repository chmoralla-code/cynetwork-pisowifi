PISOWIFI MACHINE IMAGE SETUP
=============================

The packages on the website reference a PisoWiFi machine image.

REQUIRED FILE:
- Filename: pisowifi-machine.png
- Location: This folder (assets/images/)
- Expected use: Displays on all 3 package cards

HOW TO ADD THE IMAGE:
1. Take the PisoWiFi machine PNG image you provided
2. Crop/resize it if needed (recommended: square aspect ratio, 600x600px minimum)
3. Save it as "pisowifi-machine.png" in this folder
4. Reload the website to see it display on all package cards

IMAGE SPECIFICATIONS:
- Format: PNG
- Recommended size: 600x600px or larger
- Aspect ratio: Square (1:1) works best
- File size: Under 1MB for fast loading

ALTERNATIVES:
- You can use any image you prefer (just keep the filename as pisowifi-machine.png)
- Or change the image filename in index.html and use a different name
